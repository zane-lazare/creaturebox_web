About Creaturebox Web Interface

The Creaturebox Web Interface provides a convenient way to manage and control your Creaturebox/Mothbox device directly from your web browser.

Features

  • View system status and information
  • Browse and manage captured photos
  • Control camera and lighting
  • Configure schedules and settings
  • Manage system operations

Version Information

Web Interface Version: 1.0.0
Creaturebox Version: Unknown
System: Raspberry Pi 5
